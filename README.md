# Community Detection and Hub Analysis in the Forest Gump Network

## Overview
This project explores the structure of the *Forest Gump* character interaction network using techniques from Social Network Analysis (SNA).  
The analysis focuses on identifying communities, influential hubs, and structural heterogeneity within the graph.

The project was developed in Python using NetworkX and graph visualization techniques.

---

## Objectives
- Analyze the topology of the network
- Detect communities and clusters
- Identify hubs and influential characters
- Study connectivity and structural heterogeneity
- Visualize the network and its partitions

---

## Methods
The project includes:

- Graph preprocessing
- Community detection algorithms
- Centrality analysis
- Degree distribution analysis
- Connectivity analysis
- Graph visualization with Gephi and NetworkX

---

## Technologies
- Python
- NetworkX
- Matplotlib
- NumPy
- Gephi
- Jupyter Notebook

---

## Dataset
The dataset represents interactions between characters from the movie *Forest Gump*.  
Nodes represent characters, while edges represent interactions between them.
