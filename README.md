# Case study on the Forrest Gump interaction network

## Overview
This project explores the structure of the *Forest Gump* character interaction network using techniques from Social Network Analysis (SNA).

The analysis focuses on:
- community detection,
- hub identification,
- structural heterogeneity,
- centrality analysis,
- graph visualization.

The project was developed using Python, NetworkX, and Gephi.

---

## Objectives
- Analyze the topology of the network
- Detect communities and clusters
- Identify influential hubs
- Study graph connectivity
- Visualize network structures

---

## Technologies
- Python
- NetworkX
- Pandas
- NumPy
- Matplotlib
- Gephi
- Jupyter Notebook

---

## Dataset
The dataset represents interactions between characters from the movie *Forest Gump*.

- `nodes.csv` → characters
- `edges.csv` → interactions between characters
